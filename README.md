# CGPA Calculator 

## Description
A simple C++ console application that calculates the student's GPA/CGPA based on course grades and credit hours.

## Features

- Enter any number of courses.
- Input course name.
- Input letter grade.
- Input credit hours.
- Automatic grade conversion.
- Calculates:
  - Total Credits
  - Total Grade Points
  - GPA
  - CGPA
- Displays all entered courses.
